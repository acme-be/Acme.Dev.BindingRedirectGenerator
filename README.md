And some IDE doesn't manage them ... Here is a small tool :

1. Copy the entire error message from your IDE
1. Paste into the online tool : https://binding-redirects-generator.acme-dev.be
1. Let the tool clean the error message
1. Copy the output and paste it into your .config file, in the <assemblyBinding xmlns="urn:schemas-microsoft-com:asm.v1"></assemblyBinding> section
